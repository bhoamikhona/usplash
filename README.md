# TO Use

- Clone or download the project
- use `npm install` to install all the dependencies
- create .env file
- add your Unsplash api access key as REACT_APP_ACCESS_KEY
- use command `npm start` in terminal to run the project

# Tech Used:

- React
- React Router
- React Icons
